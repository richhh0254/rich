# 资源文件说明

请在此目录下放置以下资源文件：

## 必需文件

- `icon.png` - App图标 (1024x1024)
- `splash.png` - 启动画面 (1242x2436)
- `adaptive-icon.png` - Android自适应图标 (1024x1024)
- `favicon.png` - Web favicon (48x48)

## 图标规格

- **icon.png**: 1024x1024 像素，PNG格式，无透明背景
- **splash.png**: 1242x2436 像素，PNG格式
- **adaptive-icon.png**: 1024x1024 像素，PNG格式，前景图标
- **favicon.png**: 48x48 像素，PNG格式

## 临时方案

在开发阶段，可以使用占位图片或从以下网站获取：
- [Unsplash](https://unsplash.com) - 免费高质量图片
- [Flaticon](https://www.flaticon.com) - 免费图标

## 设计建议

- 使用深色主题（黑色背景）
- 主色调：绿色 (#00ff88) 用于强调
- 保持简洁现代的设计风格
